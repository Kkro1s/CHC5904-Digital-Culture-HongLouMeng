# HTML静态网站使用说明

## 文件说明

### index.html
这是一个完整的HTML静态网站，展示了薛寶釵社交网络分析的完整内容，包括：
- 项目概览和主要发现
- 数据来源和准备过程
- 16种网络分析算法的详细说明
- 可视化结果展示
- 研究结论

## 使用方法

### 方法1：直接在浏览器中打开
```bash
# 在Finder中双击 index.html 文件
# 或在终端中运行：
open index.html
```

### 方法2：使用本地服务器（推荐）
```bash
# Python 3
python3 -m http.server 8000

# 然后在浏览器中访问：
# http://localhost:8000/index.html
```

### 方法3：用于StoryMaps

#### 步骤：
1. 访问 https://storymaps.arcgis.com/
2. 创建新的StoryMap
3. 在需要的地方添加"Web Page"或"Embed"内容块
4. 上传 `index.html` 文件到可访问的服务器（如GitHub Pages、Netlify等）
5. 将HTML文件的URL嵌入到StoryMap中

#### 或者：
1. 将HTML内容转换为StoryMap格式
2. 使用StoryMap的各个部分来展示不同内容：
   - 标题页：项目介绍
   - 内容页1：数据来源
   - 内容页2：网络分析算法
   - 内容页3：可视化结果
   - 内容页4：主要发现

### 方法4：用于Streamlit

#### 选项A：嵌入HTML内容
在Streamlit应用中嵌入HTML：

```python
import streamlit as st

# 读取HTML文件
with open('index.html', 'r', encoding='utf-8') as f:
    html_content = f.read()

# 显示HTML
st.components.v1.html(html_content, height=600, scrolling=True)
```

#### 选项B：使用现有的Streamlit应用
运行现有的Streamlit应用：
```bash
streamlit run 4_streamlit_app.py
```

## 网站内容结构

1. **项目概览** - 关键统计数据和主要发现
2. **数据来源与准备过程** - 详细的数据处理流程
3. **网络分析算法** - 16种算法的说明
4. **主要发现** - 分析结果和洞察
5. **可视化结果** - 网络图和图表
6. **工具与方法** - 使用的工具列表
7. **研究结论** - 总结和意义

## 注意事项

1. **图片路径**：确保 `data/results/` 目录下的图片文件存在：
   - `network_visualization.png`
   - `centrality_comparison.png`
   - `advanced_metrics.png`

2. **相对路径**：HTML文件使用相对路径引用图片，确保文件结构正确

3. **浏览器兼容性**：建议使用现代浏览器（Chrome、Firefox、Safari、Edge）

4. **移动端适配**：网站已包含响应式设计，可在移动设备上查看

## 部署建议

### GitHub Pages
1. 创建GitHub仓库
2. 上传所有文件
3. 在仓库设置中启用GitHub Pages
4. 访问 `https://yourusername.github.io/repository-name/index.html`

### Netlify
1. 访问 https://www.netlify.com/
2. 拖拽整个项目文件夹
3. 自动部署完成

### 其他平台
- Vercel
- Firebase Hosting
- AWS S3 + CloudFront

## 自定义修改

如果需要修改网站内容：
1. 编辑 `index.html` 文件
2. 修改相应的HTML和CSS代码
3. 保存后刷新浏览器查看效果
