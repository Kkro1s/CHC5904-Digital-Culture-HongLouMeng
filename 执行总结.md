- 薛寶釵与12个人物有直接互动
- 主要互动对象：賈寶玉（11次）、林黛玉（8次）、賈環（4次）

## 🚀 如何使用

### 快速开始
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行完整分析（如果还没运行）
./run_analysis.sh

# 3. 启动Streamlit应用
streamlit run 4_streamlit_app.py
```

### 查看结果
1. **CSV文件**: 用Excel或任何文本编辑器打开
2. **网络图**: 查看PNG图片文件
3. **Gephi分析**: 打开`network.gml`文件
4. **交互式应用**: 运行Streamlit应用
